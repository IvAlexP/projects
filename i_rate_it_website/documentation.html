Abstract

This document presents the architecture and features of the I RATE IT web application, a modular platform for managing user evaluations and opinions about various entities—such as professions, institutions, companies, products or services. Users can specify key characteristics, (un)desirable traits and leave comments. The system generates statistics and rankings, exportable as CSV or PDF, and provides an RSS feed of entity rankings. This guide serves as a reference for developers and administrators, covering structure, APIs and extensibility.

1. Introduction

I RATE IT is built using PHP, HTML, CSS and JavaScript . It enables management of entities (locations, products, services), reviews, users and roles, with data export, JWT authentication and a user-friendly responsive interface. The goal is to centralize feedback and analytics about various entities, allowing users to share their experiences and opinions effectively.

2. Features

  • See entities based on categories or your location
  • Filter and sort entities
  • Write your review from 1 to 5 stars
  • Reply to others' reviews
  • Report unadequated comments
  • Like/Dislike reviews

  • Admin Features

    • Manage users, roles, comments and entities
    • View statistics and rankings that can be exported

  • Owner Features

    • Add, edit or remove entities

10. Process

The project was developed using GitHub for a better colaboration and to ensure stability at all stages. The first pages to be created were Register and Login, followed by the Home page. The entities were then added, allowing users to see them based on categories. The next step was to implement the review system, enabling users to leave comments and ratings for entities. The admin and owner features were developed last, allowing management of users, roles, comments and entities. The idea with the admin and owner features was to allow users to have a better experience, by being able to manage their own entities and comments, while also allowing admins to manage the entire platform. We added the Near you section as a bonus feature, allowing users to see entities based on their location.
